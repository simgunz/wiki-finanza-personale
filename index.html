    Scrivere la wiki

    • La wiki è composta da file di testo in formato markdown
    • È possibile editare i file con un normale editor di testo o con un editor markdown, ad esempio Typora
    • I nomi dei file non contengono spazi e le parole sono separate da -
    • I file vengono tracciati usando git (pull - commit - push niente di più)

    Markdown

    • In questo cheatsheet si può trovare la sintassi base markdown
    • Per linkare un'altra pagina della wiki scrivere [parole con il link](nome-file-pagina.md) senza path ai file
    • Per auto-linkare un URL scrivere <https://example.com>

    Registrare

    1. Identificare il sink delle cuffie

      pactl list sinks

      In questo caso è bluez_output.5B_2C_3D_02_1A_48.a2dp-sink

    2. Identificare la source del microfono

      pactl list sources

      In questo caso è alsa_input.pci-0000_00_1f.3.analog-stereo

    3. Creare i sink virtuali

       pactl load-module module-null-sink media.class=Audio/Sink sink_name=recording sink_properties=device.description=recording
 pactl load-module module-loopback source=alsa_input.pci-0000_00_1f.3.analog-stereo sink=recording 
 pactl load-module module-combine-sink sink_name=combined sink_properties=device.description=combined slaves=bluez_output.58_FC_C6_0D_26_91.a2dp-sink

    4. Impostare Google Meet per riprodurre su combined

    5. Registrare l'audio da recording

    6. Regolare il volume del microfono e di combined per avere un volume bilanciato su recording
